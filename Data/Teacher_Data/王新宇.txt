B类研究与成果:（1）面向软件质量的符号执行和模型验证研究针对模型检测中的语言包含检查问题，尤其是对于non-Zenoness运行这一更为复杂的情况，证明了时间自动机可以为规范语言执行语言包含检查，并提出了基于CUB-TA的non-Zenoness检测算法以及基于带图的语言包含检查算法，在这一过程中引入了无限时间树展开和模拟简化等操作进一步抽象与简化问题，减少计算开支，提高检测准确性。相关研究成果发表在（TSE 2017 CCF A），（TSE 2015 CCF A）和（TACAS 2014）上。针对程序测试，基于马尔可夫决策过程模型提出了最优动态符号执行测试的策略，同时结合模型检测的相关方法研究了系统的可靠性预测以及可靠性分配问题。相关研究成果发表在（ICSE 2018 CCF A）和（ISSTA 2013）上。（2）系统质量提升方法研究针对软件缺陷的检测，定位，和修复人指派等软件生命周期中耗时、繁琐的任务，探索了与机器学习技术相结合的智能软件工程新方法，相继提出了混合模型重构算法(HYbrid moDel Reconstruction Approach HYDRA)、多特征话题模型(Multi-Feature Topic Model  MTM)，跨语言缺陷定位模型(CrosLocator)等基于软件仓库挖掘的缺陷管理方法。针对代码注释自动生成，API文档检索优化，相继提出了最近邻生成器模型(Nearest Neighbor Generator)，领域知识无关的API推荐模型(Knowledge Gap-Free API Recommendation )，探索了采用机器学习相关技术来改进软件开发过程和管理效率。研究成果在缺陷预测（TSE2017，CCFA），缺陷修复人指派（TSE2017，CCFA）、缺陷定位（ICPC2014，CCFB）、软件文档自动生成（ASE2018，CCFA），API检索（ASE2018，CCFA）等应用中取得良好效果。（3）系统质量建模与预测方法研究网络服务质量的预测，由于网络的波动性和差异性，使该任务充满挑战。针对此问题，采用一系列启发式的方法，结合时空数据、声誉感知进行服务质量预测和服务推荐。提出了时空服务质量预测(Spatial-Temporal QoS Prediction Approach)、云迁移技术设计优化(Design Optimization for Cloud Migration)、网络感知服务质量预测(network-aware QoS prediction)、Web服务的质量预测(QoS prediction of web services )，首次定义了网络感知服务组合问题，并引入空间位置数据和时序数据，对组合服务质量进行预测。研究成果在服务推荐（ACM Transactions on the Web，2016）、服务质量预测（ IEEE Transactions on Services Computing，2015）、云计算（IEEE Transactions on Services Computing  2014）等应用中取得良好效果。教学工作:《软件工程基础》《项目实训》《金融信息系统》工作研究项目:序号项目名称项目来源1基于代码大数据的程序语义学习与现场代码生成技术国家重点研发计划项目2基于时空向量回归的服务质量预测方法研究国家自然科学基金3基于云计算的物流信息服务关键技术与云平台研究与开发国家科技支撑计划项目4网构软件的可信和服务质量需求建模与分析973计划项目5基于国产高端容错计算机及国产海量存储的银行客户营销系统应用示范国家科技支撑计划项目6基于问题框架的FR/NFR关联追踪方法研究国家自然科学基金7供应链协同互联关键技术研究与应用交通运输部项目8面向个性化服务的业务流程分析与可信建模技术研究与系统开发国家科技支撑计划项目9数字图书馆-用户建模子项目国家发改委10面向分布式数据划分环境的遗留系统组件识别技术研究博士后基金项目11网络会议系统可靠性建模与研发国际合作项目12面向金融领域的可信化系统分析与构建技术研究国际合作项目13A Formal Method for Quality   Assessment of Large-Scale Software Systems新加坡技术与设计大学合作研究项目 实验室介绍:    浙江大学超大规模信息系统研究中心（软件工程研究中心）成立于2001年，充分考虑我国当前信息化发展和需求，致力于研究关键领域面临的超大规模信息系统问题，通过理论探索和工程实践的有机结合，寻求我国关键领域信息化发展的突破。     超大规模信息系统研究中心研发团队经过多年的发展，已经具备了稳定的研发梯队，其中包括教授/副教授9人，博士、硕士研究生百余人，以及本科生数十人，形成了年轻、充满朝气的研发团队。   超大规模信息系统研究中心从事大规模信息工程研究已近10年，承担了数十项国内外大规模信息系统的研发，在全球化大规模信息系统开发、系统测试、过程管理和质量保证等方面积累了丰富的科研和工程经验。针对大规模信息系统特点及面临的问题展开创新性研究，提出了多项大规模信息系统基础架构、项目管理等方面的技术与管理模型。当前研究中心的主要研究方向包括分布式计算与软件架构、超大规模数据库、金融信息系统以及软件工程。自01年来发表论文150余篇，SCI检索20余篇，其中部分成果发表在ICSE2006，ICSM2007等软件工程的国际会议与期刊上。    超大规模信息系统研究中心积极参与国内、国际交流与合作，同国内外多个院校和金融机构建立了广泛的科技交流与合作关系，如美国道富银行、英国曼彻斯特大学、美国思科公司、浙大网新等。从2001年开始，核心团队成功举办了九届“面向金融服务的信息技术国际研讨会”，与会者包括Microsoft，IBM，HP，Oracle等国际顶尖的IT公司和国际知名的金融机构代表，共达4000余人次，已经成为一个探讨全球最前沿的金融软件开发理念与技术以及开发过程的有效交流平台。
